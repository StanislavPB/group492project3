Проект «Онлайн-курсы с возможностью обучения и тестирования».

При старте программы появляется основное меню.
Есть два варианта входа в систему: как студент или как администратор.

Возможности администратора:

- вход в систему по логину;
- создать курс;
- добавить на курс студентов;
- отредактировать список студентов и конкретные обучающие материалы;
- найти/добавить материалы для обучения;
- найти/добавить тесты с вопросами для студентов;
- найти конкретный курс;
- посмотреть все курсы;
- посмотреть наполнение конкретного курса: список студентов на курсе, их успехи
(оценки, процент правильных ответов на вопросы теста) на курсе;
- посмотреть вопросы для теста на курсе;
- скорректировать название курса;
- и, наконец, удалить весь курс.

Возможности пользователя (студента):
- регистрация в системе с созданием пароля и логина;
- выбор курса с просмотром всего набора курсов;
- просмотр списка курсов, на которые студент записан;
- поиск и просмотр учебных материалов на конкретном курсе, на который студент записан;
- принять участие в тестировании, а именно: ответить на вопросы теста, предлагаемого на
конкретном курсе, к которому студент имеет доступ.

Особенности прохождения теста:
- выбор ответа на конкретный вопрос, ответы обозначены цифрой, студент выбирает цифру,
стоящую перед одним из трех или пяти вариантов ответа на поставленный вопрос;
- после прохождения теста студент видит результаты, а именно: процент правильных ответов;
- если студента данный процент не устраивает, у него есть возможность пройти тестирование еще раз.
